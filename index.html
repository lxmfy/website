LXMFy

Easily create LXMF bots for the Reticulum Network with this framework.

v0.0.0 
pip install lxmfy
lxmfy create mybot

Features

Hot Reloading

Cog system for modular bot development

Moderation

Built-in commands for unban, stats, and more

Spam Protection

Rate limiting, cooldowns, warnings, and banning

Flexible Commands

Customizable command prefix (yes you can disable it) and announcement intervals

Persistent Storage

Built-in JSON storage system with extensible backend support so you can use your own storage system

Transport Layer

Path discovery, link caching, and request handling for nomadnet nodes

Python Example

from lxmfy import LXMFBot

bot = LXMFBot(
    name="MyBot",
    command_prefix="/",
    admins=["your_hash_here"],
    announce=600
)

@bot.command(name="hello", description="Say hello")
def hello(ctx):
    ctx.reply("Hello!")

bot.run()

Framework Development

Core Architecture

Core Module

Core bot functionality, message routing and transport system.

class LXMFBot:
    def __init__(self):
        self.transport = Transport()
        self.storage = Storage()
        
    async def request_page(self, dest_hash, path):
        return await self.transport.request_page(
            dest_hash, path
        )
Key Components
  • RNS/LXMF Integration - Network handling
  • Transport Layer - Page requests and path handling
  • Storage System - Persistent data handling
  • Extension System - Hot-reloadable modules

Framework Architecture

Core systems and extension points for framework development.

Framework Components:
├── core.py         # Message routing & bot core
├── transport.py    # Extensible transport layer
├── storage.py      # Storage backends
├── commands.py     # Command system
├── moderation.py   # Spam protection
└── cogs_core.py    # Extension loading

Extension Points:
- Custom Request Handlers
- Message Pipeline
- Storage Backend
- Event System
Development Areas
  • Request Types - Custom network protocols
